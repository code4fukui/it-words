# IT用語集

IT用語を検索・閲覧できるアプリケーションです。主な機能として、キーワード検索、ランダムな用語表示などを備えています。このリポジトリには、IT用語のデータファイルと、そのデータを利用したサンプルアプリが含まれています。

## デモ
[IT辞書](https://code4fukui.github.io/it-words/)

## 機能
- キーワード検索
- ランダムな用語表示

## データ
- [it-words.csv](it-words.txt)
